# AutoFlow — Documentación

Repositorio oficial de documentación técnica y de arquitectura de **AutoFlow**, la plataforma CRM + WhatsApp para PYMEs ecuatorianas de EGIT Consultoría.

## 📁 Estructura

```
Arquitectura/
├── adr-001-stack.md          # Stack Tecnológico (v2.0) ✅ Aprobado
├── adr-002-arquitectura.md   # Arquitectura de Microservicios (v2.2)
├── api-spec.yaml             # Especificación de APIs (OpenAPI 3.0)
├── c4-context.md             # Diagramas C4 (Context + Container)
├── dev-guide.md              # Guía de Desarrollo
└── project-structure.md      # Estructura del Proyecto
```

## 🏗️ Stack Tecnológico
- **Backend:** Spring Boot 3 + Java 21
- **Mobile:** Android (Kotlin + Jetpack Compose) + iOS (SwiftUI)
- **DB:** PostgreSQL + MongoDB + Redis
- **Mensajería:** RabbitMQ
- **WhatsApp:** Evolution API
- **Push:** Firebase Cloud Messaging
- **Deploy:** Docker Compose en VPS
